  • Miraila
    Miraila是由Miraila基金会发起的基础公链项目，独创的DEPOS+PBFT共识和 DAG 分布式存储方式。
    定位下一代商用级基础公共区块链，旨在建立价值流通的信任网络，
    致力于打造广泛数字信任、价值自由流通、大众共享应用的分布式跨链商业生态。
    Miraila 具有多链、多语言、高并发、通证抽象和完整生态等特点。
  • MISS
    Miraila Token （MISS）总量100 亿枚，且承诺永不增发。
    前期Miraila Token 是基于ERC20智能合约的标准代币，
    待Miraila chain主网上线后，Miraila Token可以1:1兑换成Miraila chain。

共识算法

Miraila采用基于DPoS+VRF算法的BFT共识机制，DPoS使用去中心化的投票机制，进行超级节点的民主化选举，同时，DPoS的区块可以比 PoW 或者 PoS容纳更多的交易数量，大大提高了交易的速度。超级节点之间采用BFT共识机制，超过2/3的确认即可达成不可逆验证。

有向无环图

为了解决传统区块链中只有一条主链，无法并行执行，且存储代价高的问题，Miraila使用DAG(有向无环图)账本结构来进行数据的存储，DAG网络结构没有交易费用，而且交易确认时间短，非常适合存储大量数据。

跨链

未来一段时间内，传统互联网与区块链应用将会并存，Miraila采用多重签名技术的智能合约来实现跨链交易，适配器为主链创建多签地址，账户锁定，可适配比特币、以太坊等主流加密货币。

智能合约

Miraila 默认采用 Docker 容器化技术实现智能合约。在业界，Docker 作为智能合约的虚拟机已经在 HyperLedger Fabric 中应用，被大量的企业和项目所论证。
